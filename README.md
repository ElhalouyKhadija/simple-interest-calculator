# Simple Interest Calculator

A simple Python program that calculates **simple interest** based on user input.

## 📌 Description

This program asks the user to enter:
- Principal amount
- Annual interest rate (in percentage)
- Time (in years)

It then calculates and displays the **simple interest** using the formula:

Simple Interest = (Principal × Rate × Time) / 100

## 🚀 How to Run

1. Make sure you have Python installed (Python 3 recommended).
2. Copy the code into a file named `simple_interest.py`.
3. Open a terminal or command prompt.
4. Run the program:

```bash
python simple_interest.py
